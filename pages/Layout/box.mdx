## API Reference

This component is based on the `div` element and supports [common marginÂ props](/Variants/layout).

Only the `display` prop values are unique to the Box component.

| Prop    | Type                                            |
| ------- | ----------------------------------------------- |
| display | `none` \| `inline` \| `inline-block` \| `block` |

The props are shared between [Box](/Layout/box), [Flex](/Layout/flex), [Grid](/Layout/grid), [Container](/Layout/container) and [Section](/Layout/section) layout components.
