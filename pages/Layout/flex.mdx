## API Reference

This component is based on the `div` element and supports [common marginÂ props](/Variants/layout).

Use these props to create flex layouts.

| Prop      | Type                                                    | Default          |
| --------- | ------------------------------------------------------- | ---------------- |
| asChild   | boolean                                                 | false            |
| display   | `none` \| `inline-flex` \| `flex`                       | `flex`           |
| direction | `row` \| `column` \| `row-reverse` \| `column-reverse`  | No default value |
| align     | `start` \| `center` \| `end` \| `baseline` \| `stretch` | No default value |
| justify   | `start` \| `center` \| `end` \| `between`               | `start`          |
| wrap      | `nowrap` \| `wrap` \| `wrap-reverse`                    | No default value |
| gap       | enum                                                    |                  |

The props are shared between [Box](/Layout/box), [Flex](/Layout/flex), [Grid](/Layout/grid), [Container](/Layout/container) and [Section](/Layout/section) layout components.
