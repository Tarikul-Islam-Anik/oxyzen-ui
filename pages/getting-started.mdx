# Getting Started

Start building your project in minutes.

<Steps>
### 📦 Install the necessary dependencies

Install `class-variance-authority`, `clsx`, `tailwind-merge` utility packages by running:

```sh copy npm2yarn
npm install class-variance-authority clsx tailwind-merge
```

### 🪄 `@radix-ui` primitives

Now install necessary `@radix-ui` packages by running:

```sh copy npm2yarn
npm install @radix-ui/react-slot
```

### 📂 Get the necessary files

Now download the **components**, **lib** and **hooks** folders from the [GitHub repository](https://github.com/Tarikul-Islam-Anik/oxyzen-ui/) and paste them into your project root directory.

<Callout type="info" emoji="ℹ️">
  Click on the folder name to expand the folder.
</Callout>
