# VS Code

Here is my VS Code settings. Create a file named `settings.json` in `.vscode` folder in the root of your project and paste the following code.

```json copy filename=".vscode/settings.json"
{
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode"
}
```
